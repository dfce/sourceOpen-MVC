#   MVC 模式后端框架

## 框架应用：cookie、session、email、fs、socketIo、mysql等插件 ;
## 封装有内置：mssql数据库操作，事务模式、session,cookie读写、多文件上传，上传进度同步、即时通讯、controller权限操作控制等功能；可扩展

#   SO-1.0

### install
```
// 三张表分别应用：权限控制、事务 的测试
execute mf_localhost.sql

npm i -D
npm run start
```

### index
```
/** @desc 加载运行文件 */
const app = require('./app/app');

/** @desc 实例化应用app */
const App = new app();
// 开启RedisSession 默认本地储存session
App.allowSessionRedis();
// 开启socket.io 默认开启
App.allowSocketMode('io');

/** @desc 监听端口、开启应用 */
App.listen(8081);
```
